# tisini

[![CI](https://github.com/joshwizzy/TISINI/actions/workflows/ci.yml/badge.svg)](https://github.com/joshwizzy/TISINI/actions/workflows/ci.yml)

Mobile wallet + payments app for African SMEs and gig workers.

## Getting Started

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```

## Running Tests

```bash
flutter test
```
